a) Алгоритм Метрополиса и детальный балланс. Методом Монте-Карло найти зависимость намагниченности от температуры в одномерной модели Изинга. Сравнить полученные данные с известным из курса статистической физики аналитическим решением этой модели. б) Пропагатор пуассоновского случайного процесса удовлетворяет уравнению:
$$\partial_t T (n, t | n', t') = \gamma T (n-1, t | n', t') - \gamma T (n, t | n', t').$$
В качестве $n$ может выступать, например, число излученных фотонов при эмиссии. Решить уравнение. Численно построить соответствующий Пуассоновский случайный процесс $N(t)$.
